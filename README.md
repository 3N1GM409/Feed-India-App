# Food-Donation-App
It is a food donation app where user donate food items through this app and volunteer will further distribute the food see information from app.
<h2> Used Component </h2>
<hr>
-CardView<br>
-Room Database<br>
-Firebase<br>
-AndroidX<br>
-Used language Java
